# 🚀 Portafolio Profesional - Desarrollador Web

Portafolio moderno y elegante con animaciones profesionales, diseño responsivo y sección de proyectos destacados.

## ✨ Características

- **Animaciones profesionales**: Transiciones suaves, efectos de scroll y animaciones de carga
- **Diseño moderno**: Gradientes animados, fondos dinámicos y paleta de colores moderna
- **Responsive**: Totalmente adaptado para móviles, tablets y desktop
- **Sección de foto**: Marco elegante con efecto brillo alrededor de la foto de perfil
- **Proyectos destacados**: Grid de tarjetas interactivas con links a tus proyectos
- **Navegación suave**: Links internos con scroll smooth y navegación activa
- **Rendimiento**: Optimizado para velocidad y experiencia del usuario

## 🛠️ Tecnologías

- **HTML5**: Estructura semántica
- **CSS3**: Animaciones avanzadas, gradientes y efectos de glassmorphism
- **JavaScript Vanilla**: Interactividad sin dependencias

## 📂 Estructura

```
Portfolio/
├── index.html          # Página principal
├── styles.css          # Estilos y animaciones
├── script.js           # Interactividad
├── assets/

```

## 🚀 Configuración Rápida

### 1️⃣ Agregar tu foto

Descarga tu foto y guárdala en la carpeta `assets/` con el nombre `profile.jpg`
- Formato: JPG o PNG
- Tamaño recomendado: 600x600px o mayor
- La foto debe ser cuadrada

### 2️⃣ Ejecutar localmente

Opción A - Python:
```bash
python -m http.server 8000
# Accede a http://localhost:8000
```

Opción B - Node.js:
```bash
npx http-server
```

Opción C - VS Code:
1. Instala la extensión "Live Server"
2. Click derecho en index.html → "Open with Live Server"

### 3️⃣ Personalizar (opcional)

Edita `index.html`:
- Línea 232: Cambia `contacto@example.com` por tu email
- Cualquier texto según necesites

## 🎨 Personalizar Colores

En `styles.css`, busca `:root` y modifica:
```css
:root {
    --primary-color: #667eea;      /* Azul */
    --secondary-color: #764ba2;    /* Púrpura */
    --accent-color: #f093fb;       /* Rosa */
}
```

## 🌐 Desplegar en línea

### Netlify (Recomendado)
1. Ve a netlify.com
2. Drag & drop la carpeta Portfolio
3. ¡Listo!

### Vercel
1. Crea una cuenta en vercel.com
2. Conecta tu repositorio de GitHub
3. Despliega automáticamente

### GitHub Pages
1. Crea un repositorio en GitHub
2. Sube los archivos
3. Settings → Pages → Rama: main
4. Tu sitio estará en `https://tuusuario.github.io/portfolio`

## ⚡ Animaciones

- **Letter Drop**: Letras del título caen una por una
- **Float Up**: Foto de perfil flota hacia arriba
- **Parallax**: Fondo se mueve con el scroll
- **Fade In**: Tarjetas de proyectos aparecen al hacer scroll
- **Hover Effects**: Efectos al pasar el mouse
- **Photo Shine**: Brillo animado en la foto
